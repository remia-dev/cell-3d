# Model files

Drop the downloaded Sketchfab models here as:

- `animal.glb`
- `plant.glb`

Download each from its Sketchfab page (logged in) via **Download 3D Model → glTF**,
choosing the **`.glb`** (auto-converted glTF) option, then rename to the names above.

Both models are **CC BY 4.0** — attribution is kept visible in the app UI:

- Animal cell 2.0 — annotated in English, by montanna
  https://sketchfab.com/3d-models/animal-cell-20-annotated-in-english-0d9f7f4257224975b2ef83a283709b2f
- Plant Cell Organelles, by CVallance
  https://sketchfab.com/3d-models/plant-cell-organelles-e61e7bdf8c8449a583b364f05e70289b

The app serves files from this folder at `/models/<name>`. Until the files exist,
the viewer shows a "model not found" panel with these instructions.
